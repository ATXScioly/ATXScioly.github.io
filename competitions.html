2019 Regional Tournament

Details

The 2018 UT Austin Regional tournament will be held on February 23, 2019 at the University of Texas at Austin. The tournament will feature tests written by faculty as well as experienced former competitors. All 23 national events and the 5 state events will be run. If you have any questions, please send them to atxscioly+tournament@gmail.com.

Trial Event: Algorithm Design

Algorithm Design is a fun Python-based computer science event that involves both on-site programming (on a computer!) and an application of theoretical knowledge to a written exam. The focus of the event will be on solving programming challenges with an understanding data structures and algorithms. If you like puzzles and want to learn and practice coding, this is perfect for you! See below for rules.
